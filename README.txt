Gerador de etiquetas para encomendas Correios

== Description ==

Utilizando o plugin Mercalona será possível gerar etiquetas e declaração de conteúdo automaticamente.

Integração com Sigep em desenvolvimento.

Atualmente compatível apenas com a versão 3 ou superior do WooCommerce.

Este plugin está em fase de testes. Mantenha sempre atualizado. Sua colaboração é muito importante. Utilize este canal para suporte: 

https://feedback.userreport.com/3f0d8211-4015-4fc6-beb9-2a45981985c3/#ideas/popular

[youtube https://www.youtube.com/watch?v=J3X43hxsLFc]

== Installation ==

[youtube https://www.youtube.com/watch?v=J3X43hxsLFc]

Procure por "Mercalona" na Loja de Plugins.
Faça a instalação e ative o Plugin.
Realize a configuração na página do plugin.
Abra a listagem de pedidos e clique em "Gerar etiqueta correios"
